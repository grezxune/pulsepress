# UxStyle.md

## Brand
- Name: PulsePress
- Feeling: cinematic minimalism, confident, tactile, high signal/low noise.

## Design Tokens
- Primary background (light): `#E8E4DD`
- Primary background (dark): `#111111`
- Accent: `#E63B2E`
- Surface: `#F5F3EE` (light), translucent charcoal surface (dark)
- Main text: `#111111` (light), `#F5F3EE` (dark)

## Typography
- Sans: Space Grotesk
- Serif accent: DM Serif Display

## Layout
- Full viewport composition, centered interaction panel.
- No document scrolling; content constrained to viewport.

## Interaction
- One primary action button with magnetic hover behavior.
- Reduced-motion support disables non-essential motion.
- Distinct hover, focus, disabled, error, and loading states.

## Accessibility
- Keyboard focus visible on the primary control.
- AA-friendly contrast target for text and controls.
- Status/error messaging announced via semantic live region usage.
